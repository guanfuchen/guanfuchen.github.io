mac下截取文件夹图标

首先新建文件夹，使用command+I打开文件夹信息窗口，点击左上角图标按住command+C复制图标，然后使用预览command+V，点击第一个图标然后导出即可，像素格式为1024*1024。

mac下换取图标

选中command+I然后点击左上角的图像command+V复制即可，删除按del即可恢复默认图标。
